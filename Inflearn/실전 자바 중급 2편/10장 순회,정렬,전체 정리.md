-> 일관성 있게 자료구조를 순회해보자. -> Iterable, Iterator

## Iterable, Iterator
- Iterable: 반복 가능한
- Iterator: 반복자

```java
public interface Iterable<T> {
    Iterator<T> iterator();
}
```
단순히 Iterator 반복자를 반환한다.

```java
public interface Iterator<E> {
    boolean hasNext();
    E next();
}
```
- hasNext(): 다음 요소가 있는지 확인한다.
- next(): 다음 요소를 반환한다. 내부에 있는 위치를 다음으로 이동한다.
